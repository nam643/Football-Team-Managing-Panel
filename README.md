# Football Team Managing Panel

A JavaFX desktop application designed to manage football teams, players, and manager operations through a graphical interface.

This project follows the **MVC (Model–View–Controller)** architecture and is built using **Java and JavaFX**.

---

# Features

• Manager dashboard for team management  
• Player swapping system  
• Team dashboard overview  
• Login authentication interface  
• Error handling interface  
• Structured MVC architecture  

---

# Technologies Used

- Java
- JavaFX
- FXML
- CSS (for styling)
- MVC Architecture
